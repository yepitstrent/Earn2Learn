Conseils pour l'utilisation du code HTML

Dans Moodle, vous pouvez employer presque toutes les balises HTML afin d'obtenir la présentation que vous désirez.

Cependant, tous les scripts (Javascript, Visual Basic, etc.) seront automatiquement supprimés, pour des raisons de sécurité.

Normalement, votre texte sera contenu dans une cellule d'un tableau construit par Moodle. Veuillez donc prendre note que :

  • il n'est pas nécessaire d'utiliser les balises <head> ou <body>
  • faites attention aux balises </table> sans balise ouvrante correspondante, qui pourraient chambouler la mise en page.

Les codes des binettes seront automatiquement transformés en leur équivalent graphique et les URLs (adresses Internet) seront automatiquement converties en hyperliens.
