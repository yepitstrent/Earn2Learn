Régler les commutateurs de page

Certains « commutateurs » sont affectés aux pages Wiki pour en changer le comportement.

Commutateur :

TXTLa page contient du texte.
BINLa page contient des données binaires.
OFFLa page est inactive.
HTMLa page admet du code HTML (c'est le réglage par défaut proposé par les paramètres généraux du Wiki).
ROLecture seule.
WRÉcriture autorisée.
