Richtext HTML editoreari buruz

Eskuragarritasuna: HTML editorea eskuragarria dago

Richtext HTML editoreak zure web gunean bertan testu-editore interfaz bat ezartzen du testuak intuitiboki editatu ahal izateko, eta HTML kode normala sortzen du.

Testuari formatua emateaz gain, editore honek hainbat funtzionalitate gehigarri ditu.

Beste aplikazioetatik testua itsatsi

    Posible da beste aplikazioetatik, Word-etik adibidez, testua kopiatu eta zuzenean editorean itsastea, formatua mantenduz. Erabil itzazu "Kopiatu" eta "Itsatsi" aukerak (edo CTRL+C eta CTRL+V).

Irudiak Txertatu

    Dagoeneko web-era igo dituzun irudiak badituzu, eta URL bitartez eskuragarriak badira, irudi hauek zure testuetan sar ditzakezu "Irudia Txertatu" botoiarekin.

Taulak Txertatu

    Zure testuei egitura emateko, tresna-barrako "taula txertatu" botoia erabil dezakezu.

Lotura txertatu

    Lotura berri bat egiteko, lehenik loturaren testua sartu, ikusgai egingo dena. Gero, aukera ezazu eta tresna-barrako lotura botoia sakatu. Loturaren helburu den URL-a sartu, eta kitto!

Aurpegierak txertatu (emotikoneak)

    Ikonotxo hauek zure testuan sartzeko, tresna-barrako aurpegieren ikonoan egin klik. Hurrengo aurpegieren artean hautatzeko aukera emango dizun leiho bat irekiko da. (Bestela, dagokion kodea zuzenean sar dezakezu zure testuan eta testua bistaratzerakoan irudi bihurtuko da).
