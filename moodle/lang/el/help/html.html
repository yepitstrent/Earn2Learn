Βοήθεια για τη συγγραφή HTML

Όταν γράφετε HTML στο Moodle, είστε ελεύθερος να χρησιμοποιήσετε οποιοδήποτε HTML tag θέλετε, προκειμένου να παραχθεί το επιθυμητό αποτέλεσμα.

Ο κώδικας θα τυπώνεται κανονικά στη σελίδα μέσα σε ένα κελί πίνακα, έτσι:

  • Δεν υπάρχει λόγος να χρησιμοποιήσεις τα tags <HEAD> ή <BODY>.
  • Να προσέχεις για τα tag που δεν ταιριάζουν </TABLE> tags γιατί μπορούν να καταστρέψουν την προβολή.
