Mode HTML

Teniu les possibilitats següents:

Sense HTML
Les etiquetes HTML es visualitzen tal qual. Tot el formatatge es fa per mitjà d'etiquetes wiki. En aquest mode els filtres funcionen.
HTML segur
Es permeten algunes etiquetes HTML. Els filtres que requereixen etiquetes no funcionen.
Només HTML
Sense etiquetes wiki, només HTML. Aquesta opció permet l'ús de l'editor HTML.
