Fitxers i pàgines web

    Aquest tipus de recurs us permet enllaçar qualsevol pàgina web o qualsevol tipus de fitxer del web. També us permet enllaçar qualsevol pàgina o fitxer que hàgiu penjat a l'àrea de fitxers del curs des del vostre ordinador.

    Les pàgines web normals simplement es visualtizen, mentre que els fitxers multimèdia poden ser processats d'una manera més intel·ligent i es poden incrustar dins d'una pàgina. Per exemple, els fitxers MP3 es visualitzen per mitjà d'un reproductor en temps real integrat, igual que els fitxers de vídeo o les animacions Flash.

    Hi ha moltes opcions per visualitzar els continguts en finestres emergents, marcs, etc.

    Si el recurs és una aplicació web o un altre tipus de contingut capaç d'acceptar paràmetres, podeu triar d'enviar informació al recurs, com ara el nom de l'usuari, l'adreça de correu, en quin curs es troba, etc.
