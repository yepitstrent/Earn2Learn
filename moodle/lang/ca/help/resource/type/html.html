Pàgina web

    Aquesta mena de recurs fa molt senzill desenvolupar una pàgina web completa dins de Moodle, especialment si esteu fent servir l'editor HTML WYSIWYG de Moodle.

    La pàgina s'emmagatzema a la base de dades, no en un fitxer, i teniu molta llibertat per fer pràcticament qualsevol cosa que vulgueu amb l'HTML, inclòs Javascript.
