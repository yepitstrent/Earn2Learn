Pag-angkat ng "Format na WebCT Quiz" files

Ang WebCT na pang-angkat na filter ay pinauunlad pa at hindi sinusuportahan ang lahat ng uri ng tanong ng WebCT.

Sa panahong isinusulat ito, tanging ang Maraming Pangpipiliang Sagot at Maikling sagot na tanong ang sinusuportahan.

Narito ang ilang halimbawa ng mga tanong na ito sa isang file na teksto:

# Start of question: Short Answer Question
:TYPE:S:
:TITLE:Maikling Sagot na Tanong
:QUESTION:H
Ano ang pangalan ng WebCT Learning Hub?
:IMAGE:dotcomLogoc.gif
:ANSWERS:1
:CASE:0
:ANSWER1:WebCT.com:100:0:20:0
:FEEDBACK1:H
Bisitahin ninyo ang Learning Hub\: www.WebCT.com
:CAT:Default
# End of question: Short Answer Question
