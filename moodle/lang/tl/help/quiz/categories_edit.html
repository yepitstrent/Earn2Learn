Mga kategoriya ng tanong

Sa halip na ibuslo ang lahat ng tanong mo sa iisang mahabang listahan, maaari kang lumikha ng mga kategoriya na pagsisidlan ng mga ito.

Ang bawat kategoriya ay binubuo ng isang pangalan at maikling deskipsiyon.

Ang bawat kategoriya ay maaari ring "ilathala", alalaong baga'y ang kategoriya (at lahat ng tanong nito) ay magagamit ng lahat ng kurso sa server na ito, upang magamit din ng ibang kurso ang mga tanong mo sa kanilang mga pagsusulit.

Deskripsiyon ng mga puwang

Magulang: Ang kategoriya kung saan ilalagay ang isang ito. Kung wala pang ibang kategoriya na nalikha, tanging 'Tuktok' ang magagamit

Kategoriya: Ang pangalan ng kategoriya.

Impo ng kategoriya: Maikling deskripsiyon ng kategoriya.

Ilathala: Kung ilalathala kaagad o hindi ang kategoriyang ito.
