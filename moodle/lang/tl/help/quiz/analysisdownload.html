Pagdownload ng Pagsusuri

Marahil ay nais mong idownload ang datos na ipinapakita sa manghad na nasa screen para mas malalim na masuri ito. Maaari kang mamili sa tatlong format ng file para sa idinownload na datos.
Sa bawat klase, ang datos ay ihahandog bilang isang manghad na may angkop na pamagat, para ring nasa screen. Kung nahahati sa mga pahina ang manghad, ang lahat ng datos para sa lahat ng pahina ay idadownload sa iisang file.

Format na Excel Spreadsheet:

Makakakuha ka ng .xls spreadsheet na dokumento.

Format na OpenOffice Writer:

Ang datos ay ipapakita sa loob ng manghad na nasa OpenOffice .sxw text na dokumento.
Ang format na ito ay magagamit lamang kung ininstol mo ang PHPWriter library sa /moodle/lib na subdirektoryo.

Format na Teksto

Sa klaseng ito, ang datos ay iimbakin bilang isang regular na tekstong file. Isang linya para sa bawat row sa manghad, na ang datos ay pinaghihiwalay ng mga tabstop.
