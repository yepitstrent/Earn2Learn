Kapag nagsusulat nang online, mahirap minsang magpakita ng emosyon sa pamamagitan ng plain text. Makakatulong ang mga icon na ito.

Upang maisingit ang mga icon na ito sa inyong teksto: iklik lamang ang icon sa itaas, o itype ang kaugnay na text code sa inyong teksto.

Mapapansin na ang mga code mismo ay parang maliliit na larawan - ipaling lamang ang ulo ninyo sa kaliwa habang nakatingin sa mga ito.
