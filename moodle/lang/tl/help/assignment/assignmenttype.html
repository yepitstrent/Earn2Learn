Uri ng Takdang-aralin

May iba't-ibang uri ng mga takdang aralin

Aktibidad na offline -- maigi ito kung ang takdang-aralin ay ginagawa sa labas ng Moodle. Makakakita ang mga mag-aaral ng deskripsiyon ng takdang-aralin, pero hindi maaaring mag-aplowd ng mga file o anupaman. Magkagayunman, maaaring markahan ng mga guro ang lahat ng mag-aaral, at makikita ng mga mag-aaral ang patalastas ng mga marka nila.

Mag-aplowd ng isang file - ang uri ng takdang-aralin na ito ay nagpapahintulot sa lahat ng mag-aaral na mag-aplowd ng isang file, anuman ang uri. Maaari itong maging Word processor na dokumento, isang larawan, isang nakazip na web site, o anumang bagay na hiniling mo sa kanilang ipasa. Maaaring markahan ng mga guro ang ipinasang takdang-aralin nang online.
