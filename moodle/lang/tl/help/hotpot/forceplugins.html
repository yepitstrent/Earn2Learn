Pilitin ang Moodle ang gumamit ng mga angkop-sa-Moodle na mga media player

Hindi
Hindi babaguhin ng Moodle ang mga kaayusan ng anumang media player sa pagsusulit
Oo
Pipilitin ng Moodle ang pagsusulit na gumamit ng mga angkop-sa-Moodle na media player para paganahin ang mga file na tulad ng:
  • avi
  • mpeg
  • mpg
  • mp3
  • mov
  • wmv
