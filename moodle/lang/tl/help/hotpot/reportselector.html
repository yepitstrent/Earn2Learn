Pagpilì ng Ulat

Maaari kang magtakda ng iba-ibang pamantayan ng pagpipilian at iba-ibang format ng ilalabas para sa ulat ng Hot Potatoes.

Pagpilì ng kurso (Para lamang sa mga admin)
ang kursong ito
Ipapakita ng ulat ang mga tugon ng mga kalahok sa kasalukuyang kurso
lahat ng kurso ko
Ipapakita ng ulat ang lahat ng tugon ng mga kalahok sa lahat ng kurso na gumamit ng pagsusulit na ito at kung saan ikaw ang guro. Para magamit sa maraming kurso, kailangang nasa direktoryo ng mga file ng Site ang pagsusulit. Tanging ang mga administrador ng site ang makagagamit ng opsiyong ito.
Pagpilì ng mga user
Mga mag-aaral
Ipapakita ng ulat ang mga tugon ng mag-aaral na kumukuha ng kursong ito. Hindi isasama ang tugon ng mga guro sa ulat.
Lahat ng kalahok
Ipapakita ng ulat ang mga tugon ng mga mag-aaral, guro, tagalikha ng kurso at administrador. Ang tugon ng mga "bisitang" user, kung mayroon man, ay isasama rin.
Pagpilì ng mga marka
pinakamahusay na pagkuha
Isasama lamang ng ulat ang tugon mula sa pinakamahusay na pagkuha ng bawat user ng pagsusulit
lahat ng pagkuha
Isasama sa ulat ang mga tugon mula sa lahat ng pagkuha ng bawat user
Pagpilì ng format ng ilalabas
Kabuuang Tanaw
Ang ulat na ito ay listahan ng mga pagkuha ng pagsusulit na ito. Kasama ang mga "inaabandonang" pagkuha, nguni't ang "kasalukuyang ginagawa" na pagkuha ay hindi ipapakita. Ipapakita nito ang larawan, pangalan at pinakamahusay na iskor ng bawat user, at ang iskor, oras at tagal ng bawat pagkuha. Maaaring piliin at pagkatapos ay burahin ang mga pagkuha kung kinakailangan.
Simpleng estadistika
Ang ulat na ito ay listahan ng lahat ng nakumpletong pagkuhan ng pagsusulit na ito. Ang mga pagkuha na may tandang "inabandona" o "kasalukuyang ginagawa" ay hindi isasama. Ipinapakita ng ulat ang pangalan at pinakamahusay na iskor ng bawat user, at ang iskor ng bawat tanong sa bawat pagkuha. Ibinibigay din nito ang mga katamtamang iskor ng indibidwal na tanong at ng pagsusulit sa kabuuan. Ang detalye ng mga pagkuha ay maaaring idownload sa format na Excel o teksto.
Detalyadong estadistika
Ipinapakita ng ulat na ito ang buong detalye ng lahat ng nakumpletong pagkuha ng pagsusulit na ito. Ang mga pagkuha na may tandang "inabandona" o "kasalukuyang ginagawa" ay hindi isasama. Naglalaman ang ulat ng mga sumusunod na manghad:
  • ang manghad ng mga tugon, na nagpapakita kung paano tumugon ang bawat user sa bawat tanong sa bawat pagkuha nila ng pagsusulit.
  • Ang pagsusuri ng aytem na manghad, na naglilista ng mga tugon sa bawat tanong at sa kalimitan ng bawat tugon.
Ang mga detalye na ipinapakita sa ulat na ito ay maaaring idownload sa format na Excel o teksto.
