Format ng Ilalabas

Maaari kang magtakda ng iba't-ibang format sa pagpapakita ng pagsusulit

pinakamahusay
Ipapakita ang pagsusulit na ito sa pinakamahusay na format para sa browser.
v6+
Ang pagsusulit ay ipapakita sa isang drag at drop na format para sa mg v6+ browser
v6
Ipapakita ang pagsusulit sa format para sa mga v6 browser
v5
Ipapakita ang pagsusulit sa format para sa mga v5 browser
v4
Ipapakita ang pagsusulit sa format para sa mga v4 browser
v3
Ipapakita ang pagsusulit sa format para sa mga v3 browser
Flash
Ipapakita ang pagsusulit sa pamamagitan ng Flash
mobile
Ipapakita ang pagsusulit sa isang format na angkop sa pagtingin sa isang maliit na mobile na aparato tulad ng cell phone.
