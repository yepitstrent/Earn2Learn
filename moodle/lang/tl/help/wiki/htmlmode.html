Mode na HTML

Magagawa mo ang mga sumusunod:

Walang HTML
Iescape ang lahat ng HTML-Tag. Ang pagpoformat ay ginagawa sa pamamagitan ng mga WikiWord. Gumagana ang mga filter sa Mode na ito.
Ligtas na HTML
Pinapahintulutan ang ilang Tag at ipinapakita ito. Walang aktibong filter na mangangailangan ng tag sa mode na ito !
Tanging HTML
Walang WikiWord, HTML lamang. Pinapahintulutan ng opsiyong ito ang paggamit ng HTML-Editor.
