File ng mga pakete

Ang pakete ay isang partikular na file na may zip (o pif) na ekstensiyon na naglalaman ng tanggap na file ng AICC o SCORM na depinisyon ng kurso.

Ang isang paketeng SCORM ay dapat mayroon file na may pangalang imsmanifest.xml sa root ng zip file na nagbibigay ng depinisyon ng balangkas ng kurso, lokasyon ng rekurso, at marami pang iba hinggil sa SCORM.

Ang paketeng AICC ay binibigyang depinisyon ng ilang file (mula 4 hanggang 7) na may mga itinakdang ekstensiyon. Narito ang mga kahulugan ng mga ekstensiyon:

  • CRS - Course Description file (kailangan)
  • AU - Assignable Unit file (kailangan)
  • DES - Descriptor file (kailangan)
  • CST - Course Structure file (kailangan)
  • ORE - Objective Relationship file (opsiyonal)
  • PRE - Prerequisites file (opsiyonal)
  • CMP - Completition Requirements file (opsiyonal)
