TULONG sa pagsusulat ng html

Kapag nagsusulat ka ng HTML sa Moodle, maaari mong gamitin ang anumang HTML tag na gusto mong gamitin upang makalikha ng nais mong epekto.

Tandaan na hindi pinahihintulutan ang pagsusulat ng script (hal. Javascript o VB Script), at ang mga ito ay awtomatikong tatanggalin.

Ang code mo ay karaniwang malalathala sa pahina sa loob ng table cell, kaya:

  • hindi na kailangang gumamit ng <head> o <body> na tag
  • mag-ingat sa mga </table> tag na walang kapares, na maaaring gumulo sa displey.

Ang mga smiley (emoticon) ay ikukumberte sa katumbas nitong larawan, at ang mga payak na URL ay ikukumberte sa link.
