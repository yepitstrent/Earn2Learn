Mga Password

Itinatakda sa puwang na ito ang password para "makapasok" sa workshop. Ang password ay maaaring hanggang 10 titik ang haba. Ang password ay maaaring baguhin anumang oras sa panahon ng workshop. Itinatakda naman ng "Gumamit ng password" na opsiyon kung hihilingin ang mag-aaral na magbigay ng password bago makapasok sa workshop.
