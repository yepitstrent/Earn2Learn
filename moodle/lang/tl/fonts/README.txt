Mga Font
--------

Ang direktoryong ito ay naglalaman ng mga font na ginagamit sa paglikha
ng mga larawan na may teksto.

Ang tanging ginagamit sa kasalukuyan ay default.ttf

Kapag walang font ang wika dito ang gagamitin ay ang nasa 
/lang/en/fonts/default.ttf.


Ang mga multibyte na string ay kailangang idecode, dahil ang mga
Truetype na routine ay umaasa na may ISO font o Unicode string.  Kung
may file dito na may pangalang lang_decode.php, na naglalaman ng
function na lang_decode(), ito ay gagamitin sa bawat string.
