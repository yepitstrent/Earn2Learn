Respostas Individuais

Esta tabela mostra as respostas de cada usuário a cada pergunta em todas as suas tentativas.

The responses are color-coded as follows:

  • Respostas corretas
  • Respostas ignoradas (quando presentes)
  • Respostas erradas (quando presentes)
  • resultado% (dicas,indícios,controles)
    • resultado - calculado como porcentagem para cada pergunta
    • dicas - número de dicas pedido
    • indício - indica se o indício foi mostrado (=1) ou não (=0)
    • controles - número de controles aplicados nesta pergunta. O valor igual a "-1" significa que o estudante clicou o botão "Mostrar Resposta"
