Использование HTML редактора

При редактировании страниц эта опция разрешает использовать HTML редактор (далее редактор) вместо простого текстового поля. Редактор позволяет выполнять элементарное форматирование текста и добавлять "различные функции" такие как например ссылки.

Редактор включается путем шелчка по опции Использовать редактор. Редактор связан с выбором для определенного поля ввода. К примеру, включив редактор для поля Отзыв 1, редактор включится только для этого поля, а не для всех полей.

Если необходимо включить редактор для некоторых полей ввода, сначала выберети их путем щелчка по опции Использовать редактор(В них должны появиться галочки.) Далее страница должна быть обновлена для активации редактора. Это делается путем нажатия на кнопке Обновить страницу в нижней части.

Заметьте, что редактор необходимо использовать с осторожностью. В большинстве случаев его использование не приводит к проблемам. Тем не менее, не рекомендуется его использование с ответами к вопросам типа "Короткий ответ" и числового типа. Редактор может добавить дополнительные "невидимые" HTML тэги, кторые препятствуют функции сравения работать правильно. В дополнение Заголовки в таблице содержания используемые впоследствии в кнопках могут содержать лишние HTML тэги, что прведет к проблемам. Нет проблем с ответами, кторые используются только для показа, к примеру такие как в типе вопроса "Множественный выбор". Использование редактора с Отзывами к вопросам не должно причинять проблем.

Если возникла проблема с текстом, редактор можно выключить (путем снятия опции Использовать редактор). Когда страница обновлена, "сырой" текст может быть просмотрен в текстовом поле. Текст может быть так же отредактирован, если необходимо. На самом деле включение и отключение редактора для определенного поля ввода не должно отражаться на тексте.

"Статус" опции Использовать редактор сохраняется с каждым измененным значением тектового поля. Таким образом, при перередактировании страницы, текст набранный в редакторе будет показан в редакторе.

В дополнение, редактор требователен к ресурсам браузера. Если нет необходимости его использования, рекомендуется использовать стандартные тектовые поля ввода.
