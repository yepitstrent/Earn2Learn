Coste del Curso

Si el sitio ha sido configurado para emplear algún método de matriculación que requiere realizar un pago (como el del módulo PayPal), puede introducir el coste del curso aquí sin ningún símbolo (la moneda es configurada en el módulo de matriculación). Por ejemplo, 19.95.

Si el campo de coste está vacío, la opción de pago no se muestra y el sitio mostrará otros tipos de matriculación (como el de claves, o la matriculación manual).

Si el campo de coste NO está vacío, a los alumnos que se intenten matricular se les mostrará la opción de realizar el pago para entrar.

Si ADEMÁS configura el curso con una clave de matriculación, los estudiantes tendrán también la opción de matricularse utilizando dicha clave. Esto es útil si tiene una mezcla de alumnos de pago y no pago.
