Zeitbegrenzungen

In der Basiseinstellung haben die Teilnehmer/innen unbegrenzt Zeit zum Abschluss eines Tests nachdem sie ihn einmal begonnen haben.

Wenn Sie eine Zeitbegrenzung festlegen passiert folgendes:

  • Javascript Unterstützung im Browser ist obligatorisch, damit die Zeitfunktion korrekt läuft.
  • Ein Zeitfenster wird eingeblendet und zeigt die verbleibende Zeit.
  • Wenn die Zeit abgelaufen ist, wird der Test automatisch gespeichert und im letzten Zustand der Beantwortung abgegeben.
  • Wenn ein/e Teilnehmer/in versucht zu schummeln und mehr als 60 Sekunden über die zugewiesene Zeit verwendet, dann wird der Test automatisch mit Null bewertet.
  • Wird der Test vor dem Versuch verlassen - wird die Zeit am Server weitergezählt!!!
