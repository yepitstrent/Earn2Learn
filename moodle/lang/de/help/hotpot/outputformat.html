Ausgabeformate

Sie können die festgelegten Formate für die Tests festlegen:

beste
Der Test wird im besten Format des Browsers angezeigt.
v6+
Der Test wird im "drag and drop"-Format des für Browser in der Version 6+ angezeigt.
v6
Der Test wird für Browser in der Version 6 angezeigt.
v5
Der Test wird für Browser in der Version 5 angezeigt.
v4
Der Test wird für Browser in der Version 4 angezeigt.
v3
Der Test wird für Browser in der Version 3 angezeigt.
Flash
Der Test wird im Flash-Format angezeigt.
mobil
Der Test wird in einem Format angezeigt, dass mobile Geräte, wie z.B. PDAs unterstützt.
