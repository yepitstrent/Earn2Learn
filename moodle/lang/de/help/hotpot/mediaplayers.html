Relative URLs, MediaPlayer und Hot Potatoes

Definitionen

URL
die Adresse einer Internetquelle, wie z.B. eine Webseite, ein Bild oder eine Sounddatei.
absolute URL
eine URL, die mit "http://" beginnt, wird absolute URL genannt. Wann immer sie benutzt wird, wird Bezug auf das gleiche Ziel genommen.
relative URL
eine URL, die nicht mit "http://" beginnt, wird relative URL genannt. Sie kann gemeinsam mit der absoluten URL der Basisseite verwendet werden, um die absolute URL der Zieldatei zu bekommen.

Relative URLs in Hot Potatoes Tests

Relative URLs sind ein bequemer Weg, um Bilder, Sounds und Videos in einem Hot Potatoes-Test verfügbar zu machen. Sie ermöglichen einen Test am heimischen PC, bevor sie auf eine Moodle-Webseite hochgeladen werden. Wird der Test auf einen Moodle-Server verschoben, so müssen auch die Multimediadateien auf die Moodleseite hochgeladen werden, um das Funktionieren des Tests zu gewährleisten.

Aus Sicherheitsgründen ermöglicht Moodle nicht den direkten Zugriff auf die Kursdateien. Alle Dateianfragen innerhalb eines Kurses werden durch ein Sicherheitsskript überrpüft, um festzustellen, ob die Anfrage von einer Person gestellt wurde, die in dem Kurs eingeschrieben ist. Dieser Vorgehensweise kann relative URLs, die in den Tests benutzt werden, unbrauchbar machen .

Die Lösung des Problems ist einfach: Alle relativen URLs müssen in absolute Urls umgewandelt sein, wenn der Test den Browser erreicht. Im Falle der Hot Potatoes-Tests werden die URLs durch Moodle konvertiert, so dass sich Testersteller keinerlei Sorgen um das Funktionieren ihres Kurses machen müssen. 

Mediaplayer in Hot Potatoes-Tests

Einige Mediaplayer werden die Mediendateien nicht akzeptieren, wenn sie von dem Sicherheitsskript kommen, das die Kurse absichert. In diesem Fall ist es notwendig den MediaPlayer durch einen Moodle-kompatiblen MediaPlayer zu ersetzen. Dies können sie selber machen, indem sie die MediaPlayer-Einstellung bei den Testeinstellungen anpassen. Sie erzwingen den Gebrauch des Moodle-kompatiblen Players, indem sie die Option "Erzwinge Media-Plugin" auf "Ja" stellen.
