Navigation

Sie können unterschiedliche Navigationshilfen für ein Hot Potatoes Quiz vorgeben.

Moodle Navigationleiste
Die Standardnavigationsleiste wird in dem selben Fenster des Tests angezeigt. Die Leiste erscheint am oberen Rand des Tests.
Moodle Navigationrahmen
Die Standardnavigationsleiste wird ine einem separaten Rahmen am oberen Rand des Tests angezeigt. 
Embedded <IFRAME>
Die Standardnavigationsleiste wird im selben Fenster wie der Test angezeigt. Sie wird in einem  <IFRAME> eingebettet sein.
Hot Potatoes Test-Schaltfläche
Der Test wird mit einer Navigationsschaltfläche angezeigt werden.
Eine "Abgeben"-Schaltfläche
Der Test wird mit einer "Abgeben"-Schaltfläche am oberen Seitenrand angezeigt. 
Nichts
Der Test wird ohne Navigationshilfen angezeigt. Wenn alle Fragen richtig beantwortet wurden, wird Moodle entweder zur Kursseite zurückkehren oder das nächste Quiz anzeigen. Dies hängt davon ab, was sie bei der Wahlmöglichkeit "Nächsten Test zeigen?" eingestellt haben.
