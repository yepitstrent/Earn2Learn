パッケージファイル

パッケージは、AICCまたはSCORMコース定義ファイルを含む、zip ( または pif ) 拡張子の付いたファイルです。

SCORMパッケージは、 SCORMコース構造、リソースの場所、そのた多くを定義したimsmanifest.xmlという名称のルートzipファイルを含む必要があります。

AICCパッケージは、拡張子によって分けられたいくつかのファイル ( 4から7 ) によって定義されます。下記に拡張子の意味を記します。

  • CRS - コース定義ファイル ( 必須 )
  • AU - 割り当てユニットファイル ( 必須 )
  • DES - 記述ファイル ( 必須 )
  • CST - コース構造ファイル ( 必須 )
  • ORE - オブジェクト関連ファイル ( 任意 )
  • PRE - 前提条件ファイル ( 任意 )
  • CMP - 完結要件ファイル ( 任意 )
