Anexos para envio

Se pretender, pode anexar UM ficheiro do seu computador a qualquer envio dos fóruns. Este ficheiro é enviado para o servidor e guardado junto dos seus.

É útil quando quizer partilhar, por exemplo, uma imagem ou um documento Word.

Este ficheiro pode ser de qualquer tipo, no entanto recomenda-se veementemente que o mesmo tenha um nome utilizando os sufixos de 3-letras da Internet, tais como "doc" para um documento Word, "jpg" ou "png" para uma imagem, etc. Isto irá facilitar aos outros a obtenção e a vizualização do seu anexo.

Se reeditar uma anotação e anexar um novo ficheiro, então quaisquer ficheiros previamente anexados para esse envio serão substituídos.

Se reeditar uma anotação com um anexo, deixando este espaço em branco, o anexo original será retido.
